# Python-lxns

## Python library to control Linux kernel namespaces

Alpha version.

Implemented using C extension module.

Raw calls are available under `lxns.os` module.

## Requirements

## Compiling source package

* Meson build system
* Python headers
* C compiler
* Linux kernel headers
